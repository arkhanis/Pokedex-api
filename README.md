# Project: Pokedex-api

## About the project 🌟
### [Live Demo](https://arkhanis.github.io/Pokedex-api/)
### `Description:` 
This project is about Pokemon API consumption and type segmentation.
### `Features:`
- API consumption.
- Filters by pokemon type. 
- Responsive for all devices.
- Build using HTML, CSS, and JavaScript.
